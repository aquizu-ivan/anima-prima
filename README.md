# Ánima Prima

Obra digital contemplativa — pulso inicial del Octavo Arte IAQUIZU.

Ánima Prima es el pulso inicial: una experiencia web mínima, silenciosa y rítmica
sobre la presencia previa a toda forma. No es un portfolio ni una landing, sino
una obra diseñada para ser mirada con calma, en pantalla única, como parte del
universo IAQUIZU.
